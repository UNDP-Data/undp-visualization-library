### graphClassNames

You can use the graphClassNames object to assign custom CSS class names to specific parts of a chart.
This allows you to style those elements via external stylesheets instead of inline styles.

#### Example

```tsx
graphClassNames={{
  title: 'custom-title',
  description: 'custom-description',
  tooltip: 'custom-tooltip',
  xAxis: {
    labels: 'custom-x-axis-labels',
    gridLines: 'custom-x-axis-grid',
    title: 'custom-x-axis-title',
    axis: 'custom-x-axis-line',
  },
  yAxis: {
    labels: 'custom-y-axis-labels',
    gridLines: custom-y-axis-grid',
    title: 'custom-y-axis-title',
    axis: 'custom-y-axis-line',
  },
  mouseOverLine: 'custom-hover-line',
  regLine: 'custom-regression-line',
}}
```
