### dashboardLayout

`required`

Type - `object`

Defines the layout of the dashboard, charts that are shown and the data used to visualize these charts.

#### Example

```
dashboardLayout={{
  title: 'Sample Dashboard',
  description: 'Graph Description',
  backgroundColor: true,
  padding: '1rem',
  language: 'en',
  rows: [
    {
      height: 700,
      columns: [
        {
          graphType: 'horizontalBarChart',
          graphDataConfiguration: [
            { columnId: 'Country', chartConfigId: 'label' },
            { columnId: 'Value', chartConfigId: 'size' },
          ],
          settings: {
            graphTitle: 'Horizontal Bar Chart',
          },
        },
      ],
    },
  ],
}}
```
