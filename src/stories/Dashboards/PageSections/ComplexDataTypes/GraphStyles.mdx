### graphStyles

You can apply custom CSS styles to various parts of the chart using the StyleObject. Each key accepts a React.CSSProperties object (inline CSS style).

#### Style Targets

  <thead>
    <tr>
      <th>Key</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>

      <td>Style for the chart title.</td>

        <code>footnote</code>
      </td>
      <td>Style for the chart footnote text.</td>
    </tr>
    <tr>
      <td>
        <code>source</code>
      </td>
      <td>Style for the data source label.</td>
    </tr>
    <tr>
      <td>
        <code>description</code>
      </td>
      <td>Style for the chart description block.</td>
    </tr>
    <tr>
      <td>
        <code>graphBackground</code>
      </td>
      <td>Style for the background of the graph container.</td>
    </tr>

        <code>tooltip</code>

      <td>Style for tooltip popups.</td>
    </tr>

      <td>
        <code>xAxis</code>
      </td>

      </td>
    </tr>
    <tr>
      <td>
        <code>yAxis</code>

      </td>
    </tr>

      <td>
        <code>graphObjectValues</code>

#### Example

```tsx
graphStyles={{
  title: { fontSize: '20px', fontWeight: 'bold' },
  description: { color: '#666', fontStyle: 'italic' },
  xAxis: {
    labels: { color: '#333', fontSize: '12px' },
    gridLines: { stroke: '#ddd' },
    title: { fontWeight: '600', fontSize: '14px' },
    axis: { stroke: '#222' },
  },
  yAxis: {
    labels: { color: '#444' },
    gridLines: { strokeDasharray: '4' },
    title: { color: '#000', fontSize: '13px' },
    axis: { strokeWidth: 1 },
  },
}}
```
