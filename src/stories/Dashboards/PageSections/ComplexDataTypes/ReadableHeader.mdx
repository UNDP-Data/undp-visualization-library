### readableHeader

Type - `Array<object>`

Optional array of header definitions with display-friendly labels.

##### Data formats

<table>
  <thead>
    <tr>
      <th>Key</th>
      <th>Required</th>
      <th>Type</th>
      <th>Default value</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>value</td>
      <td>yes</td>
      <td><code>string</code></td>
      <td></td>
      <td>The internal identifier for the header (how the column is called in the csv or data source).</td>
    </tr>
    <tr>
      <td>label</td>
      <td>yes</td>
      <td><code>string</code></td>
      <td></td>
      <td>The user-friendly name shown in the UI.</td>
    </tr>
  </tbody>
</table>

##### Example

```tsx
readableHeader={[
  {
    value: 'countryCode',
    label: 'Country Code'
  },
  {
    value: 'countryName',
    label: 'Country Name'
  },
]},
```
