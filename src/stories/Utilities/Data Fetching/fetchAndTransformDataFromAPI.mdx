# fetchAndTransformDataFromAPI

Fetches and transforms data from an API, and applies optional transformations.

```ts
function fetchAndTransformDataFromAPI(
  requestURL: string,
  headers?: any,
  columnsToArray?: {
    column: string;
    delimiter?: string;
  }[],
  dataTransformation?: string,
  debugMode?: boolean,
): Promise
```

## Parameters

- `requestURL` - The URL of the API to fetch data from.
- `headers` - Optional headers to include in the API request.
- `columnsToArray` - Optional columns configuration to transform certain columns to arrays.
- `dataTransformation` - Optional Handlebars template for data transformation.
- `debugMode` - Optional flag to log data to the console for debugging.

## Returns

A promise that resolves to the transformed data.
