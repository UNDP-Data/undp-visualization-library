# fetchAndParseCSV

Fetches a CSV file from a URL, parses it, and applies transformations based on the provided configuration.

```ts
function fetchAndParseCSV(
  dataURL: string,
  dataTransformation?: string,
  columnsToArray?: {
    column: string;
    delimiter?: string;
  }[],
  debugMode?: boolean,
  delimiter?: string,
  header?: boolean,
): Promise
```

## Parameters

- `dataURL` - The URL of the CSV file to fetch.
- `dataTransformation` - Optional Handlebars template for data transformation.
- `columnsToArray` - Optional columns configuration to transform certain columns to arrays.
- `debugMode` - Optional flag to log data to the console for debugging.
- `delimiter` - Optional delimiter used in the CSV file (default is comma).
- `header` - Optional flag to specify whether the CSV file contains headers (default is true).

## Returns

A promise that resolves to the parsed and transformed data.

## Example Usage

```js
// Example of how to use the function
fetchAndParseCSV('https://example.com/data.csv')
    .then(parsedData => console.log(parsedData))
    .catch(error => console.error('Error:', error));
