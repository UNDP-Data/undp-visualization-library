# svgDownload

Downloads an SVG representation of a specified HTML element.

Converts the HTML element into an SVG format using the `dom-to-svg` library, serializes it to a string, and creates a downloadable `.svg` file.

```ts
function svgDownload(node: HTMLElement, filename: string)
```

## Parameters

- `node` - The HTMLElement to capture and convert into an image.
- `filename` - The desired filename for the downloaded image (without extension).

## Example Usage

```js
// Example of how to use the function
const element = document.getElementById('myElement');
svgDownload(element, 'downloaded-svg');
